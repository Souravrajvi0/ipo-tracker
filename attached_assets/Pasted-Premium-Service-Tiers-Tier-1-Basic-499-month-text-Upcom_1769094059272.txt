API Endpoints (Premium Service)
Authentication
text
POST /v1/auth/token
Body: {api_key, api_secret}
Response: {token, expires_in}
Tier 1 & 2 Endpoints
Endpoint	Calls/Day	Response	Use Case
GET /v1/ipos/upcoming	10	Array of next 10 IPOs	Browse upcoming
GET /v1/ipos/:symbol/live	100	Real-time sub %, GMP, bid count	Live dashboard
GET /v1/ipos/:symbol/history	100	Historical returns, performance	Analysis
GET /v1/ipos/performance	50	All IPOs + listing gains, 1M/6M ret	Portfolio tracking
GET /v1/portfolio/:user_id/pnl	100	Your holdings P&L real-time	Track investments
POST /v1/alerts/subscribe	10	Subscribe to Telegram/Email alerts	Set thresholds
Tier 3 (Enterprise) Additions
Endpoint	Response	Use Case
GET /v1/ipos/sentiment	Bullish/Bearish score, Reddit/Twitter mentions	Market mood
GET /v1/ipos/backtest	Historical scoring accuracy	Validate models
POST /v1/webhooks/subscribe	Subscribe to real-time events	App integration
GET /v1/ipos/export/csv	Bulk data export	Reporting
GET /v1/market/analysis	Peer comparison, valuation trends	Research
Revenue Model
text
Recurring SaaS:
├─ Basic: ₹499 × 1,000 users = ₹4.99L/month

Partnerships with fintech platforms

Technical Stack (Paid Service)
text
Backend: Node.js + Express
Auth: JWT + API keys (database)
Rate Limiting: Redis (per key, per tier)
Database: PostgreSQL (user data, keys, usage)
Caching: Redis (expensive API responses, webhook queue)
Queue: BullMQ (webhook delivery, scraper jobs)
Monitoring: Prometheus + Grafana (uptime tracking)
CDN: CloudFlare (if scaling)
Deploy: Docker on AWS/GCP (production reliability)
